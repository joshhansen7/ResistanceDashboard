Methodology

What's tracked
News coverage of data-center development, operations, and opposition across the United States.
Data sources
Primary ingestion is web search. A per-state sweep pipeline runs a fixed set of six high-signal query templates against each of the 50 states plus DC (≈300 searches per sweep), ensuring every state has recurring coverage rather than relying on whichever stories a single query happens to surface. Web search results are deduplicated by URL and passed through the same classification pipeline as RSS-sourced articles. A small number of curated RSS feeds (primarily regional Wyoming, Texas, and Michigan outlets) are also polled but currently contribute a minority of the corpus.
